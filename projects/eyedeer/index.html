EyeDeer

Description

An OAuth2 service provider and personal information manager. Integrates with other websites to provide user authentication and much more.

Current Status

It's currently in active development and no release has been made. It's expected that a release will be ready in a month.

Technologies Used

Typescript/NodeJS for its backend. EJS and Petite-Vue for its front end. Express for the server.
